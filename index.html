Two-Factor Authentication Setup

Android Setup

  1. Install Google Authenticator from Play Store
  2. Open the app on your target account
  3. Go to account settings ‚Üí Security ‚Üí 2FA
  4. Scan the QR code with Authenticator
  5. Enter the 6-digit code to verify
  6. Save backup codes in a secure location

iOS Setup

  1. Download Google Authenticator from App Store
  2. Open Settings ‚Üí Passwords ‚Üí AutoFill Passwords
  3. Enable 'Use with apps and websites'
  4. In target app, go to Security settings
  5. Enable 2FA and scan QR code
  6. Store backup codes in Notes app (locked)

Essential Security Tools

Password Managers

  • Android: Google Password Manager (built-in)
  • iPhone: iCloud Keychain (built-in)
  • Cross-platform: Bitwarden, 1Password

Secure Messaging

  • Signal: End-to-end encryption
  • WhatsApp: Built-in encryption (verify contacts)
  • Telegram: Secret chats only

App Permissions

  • Android: Settings ‚Üí Apps ‚Üí Permissions
  • iPhone: Settings ‚Üí Privacy & Security
  • Rule: Only grant necessary permissions

Account Monitoring

  • Google: myaccount.google.com/security
  • Apple: appleid.apple.com/sign-in
  • Facebook: Settings ‚Üí Security and Login

Advanced Security Resources

  • Monthly password strength audits
  • Dark web monitoring services
  • Account activity reviews
  • Device access logs
  • Trusted VPN providers: NordVPN, ExpressVPN
  • Always-on VPN for mobile devices
  • Avoid free VPN services
  • Use VPN on public Wi-Fi
  • Signal for sensitive conversations
  • WhatsApp disappearing messages
  • Verify contact security codes
  • Avoid SMS for sensitive info
  • Cloud backup: Google Drive, iCloud
  • Local backup: External drives
  • Photo backup automation
  • Document version control
